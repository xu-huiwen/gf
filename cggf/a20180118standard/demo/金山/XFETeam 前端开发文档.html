HTML

DOCTYPE 声明

  • XHTML 1.0 Strict DTD

    <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">

  • XHTML 1.0 Transitional DTD

    <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">

  • XHTML 1.0 Frameset DTD

    <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Frameset//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-frameset.dtd">

  • HTML5 DOCTYPE 

    <!DOCTYPE HTML>

Xmlns

Xmlns 是 XHTML namespace 的简写

  <html xmlns="http://www.w3.org/1999/xhtml">

( 如果为 HTML5 DOCTYPE 声明方式可省略此项！ )

字符集编码

  <meta charset="utf-8" />

( 注意：文件要求无BOM签名！ )

标签语义化

favicon icon

  <link rel="shortcut icon" type="image/x-icon" href="/favicon.ico" />

Author、Copyright

  <meta name="author" content="西山居游戏" /> 
  <meta name="copyright" content="www.xoyo.com, 版权所有" />

Description

  <meta name="description" content="说明性内容" />

Keywords

  <meta name="keywords" content="关键词" />

标签闭合

  • 闭合标签

    即以 <标签名> 开始 以 </标签名> 结束的 【成对】 出现的标签

    例：div p 等 要求必须有完整的结束闭合标签 </div> </p>

    <div>-----内容-----</div>
<p>-----内容-----</p>

  • 自闭合标签

    即在标签尾部以 /> 结束的标签

    例：img input 等 必须以 /> 自闭合结束

    <img alt="图片" src="xxx.jpg" />
<input type="text" name="text" value="" />

标签小写

HTML 标签要求采用 E 文小写字母

  • 推荐

    <div id="header" style="width:500px"></div>

  • 不推荐

    <DIV ID="HEADER" STYLE="WIDTH:500PX;"></DIV>

代码缩进

保持 4 个空格缩进

  <!DOCTYPE html>
  <html>
  <head> 
      <meta charset="UTF-8">
      <meta name="keywords" content="" />
      <meta name="description" content="" />
      <title>西山居游戏</title>
  </head>

  <body>
      <h1>西山居游戏</h1>
  </body>
  </html>

元素的属性值

元素的属性值都必须放置在英文状态下的 "" 中

  • 推荐

    <img alt="图片" height="80" width="100" src="xxx.png" />

  • 不推荐

    <img alt=图片 height=80 width=100 src=xxx.png />

给标签元素的属性赋值

给予了某个标签一个属性 原则上这个属性必须有一个值( 一个有意义的值 ) 这个值可以等于该属性

例：表单元素( 常常容易被大家忽略 )

  • 正确写法

    <input type="text" name="account" value="" />

  • 不规范写法

    <input type="text" name="" value="" />

为含有描述性表单元素添加 label

为含有描述性文本的表单元素添加 label 并合理的使用 for accesskey 等属性 ( accesskey 属性所设置的快捷键不能与浏览器的快捷键冲突 否则将优先激活浏览器的快捷键 )

  • 推荐

    <input id="male" type="radio" name="sex" value="" /><label for="male" accesskey="M">男</label>
<input id="female" type="radio" name="sex" value="" /><label for="female" accesskey="F">女</label>

  • 不推荐

    <input id="male" type="radio" name="sex" value="male" />男
<input id="female" type="radio" name="sex" value="woman" />女

img 的 alt 属性

针对专用于图像标签的 alt 属性 该属性是用来给图像提供一个替换文字说明 不管我们给不给图像文字说明 这个属性都应该存在于图像标签 <img> 中 如果实在是没有对应的替换文字 可以使 alt 这个属性为空值( 不是没有值 而是使用一个空的值 )

  • alt 有值

    <img alt="西山居游戏" src="xsj-logo.png" />

  • alt 空值

    <!-- 注意：引号中间没有空格！-->
<img alt="" src="xxx.png" />

链接

  • 为 <a> 标签书写 title 属性

    <a href="http://www.xishanju.com/" title="西山居游戏">西山居游戏</a>

    ( 特别是针对文字链接 有限的文字未能完全表达链接的完整信息 给链接写上 title 属性 并在 title 属性中完整输出要表达的文字内容 可以让浏览者在访问链接时看到完整的信息 )

    例：

    <a href="#more" title="点击查看更多视频" target="_blank">更多</a>

  • 非涉及页面 URL 跳转的链接推荐写法

    <a href="javascript:;"></a>
<a href="javascript:void(0);"></a>

nofloow

为非本站相关的链接 需要屏蔽搜索引擎及外部链接适当的添加 nofollow 属性

例：

  <a rel="nofollow" target="_blank" href="http://www.sznet110.gov.cn/">公共信息安全网络监察</a>

特殊字符采用实体编号或实体名称

对一些特殊的符号和字符建议采用实体编号或实体名称

例：

  • < 用 &lt; 或 &#60;

  • > 用 &gt; 或 &#62;

  • CopyRight © 中的 © 用 &copy 或 &#169;

    实体名称和实体编号均可 推荐采用实体编号 ( 使用实体名称而不是实体编号的好处在于 名称相对来说更容易记忆 而这么做的坏处是 并不是所有的浏览器都支持最新的实体名称 然而几乎所有的浏览器对实体编号的支持都很好 )
